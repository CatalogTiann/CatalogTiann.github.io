deque双端队列

发表于 2017-08-06 | 分类于 Python |
字数统计: 1.2k | 阅读时长 ≈ 5

deque 简介

deque (double-ended-queue) 是一种数据结构, 该结构和list类似, 可以看作是 list 结构的增强版, 且比列表提供了更强大的方法. 简单来说, deque 可以看作是一个双向列表, 支持左右两端的操作

下面就通过几个小例子来详解 Python 的 collections 模块中的 deque 双端队列结构

骚操作: deque 在生成双端队列时, 可以指定 maxlen 值, 如果队列内的数据条数等于 maxlen 的时候, 再插入数据时, 会把最老的数据从双端队列中剔除掉. 就类似于 redis 的 FIFO队列

deque 的操作

append()

和普通列表的 append 方法一样, 作用是向列表右端添加一个元素.

appendleft()

顾名思义, 向列表左端添加一个元素, 这时候就凸显出双端队列的强大了, 不仅可以向右端添加元素, 还可以向左端添加元素.

clear()

清空当前双端队列

copy()

浅拷贝当前双端队列, 注意, 是浅拷贝!!

count(value)

计算 value 在当前双端队列中的总个数

extend(iterable)

使用可迭代对象扩展当前双端队列(向右端扩展)

extendleft(iterable)

使用可迭代对象扩展当前双端队列(向左端扩展)

注意和上面唯一不同的就是方向

index(value, start=None, stop=None)

查询 value 第一次出现的索引值, 可以指定索引起始位置和结束位置

insert(index, p_object)

在索引值为 index 前面插入 p_object 元素

pop()

从双端队列的右端”弹出”一个值, 并将该值作为返回值返回.

popleft()

从双端队列的左端”弹出”一个值, 并将该值作为返回值返回.

remove(value)

从双端队列中移除 value 元素

reverse()

反转当前双端队列

rotate()

这是一个神奇的方法, 但是可能是因为我才疏学浅, 觉得这个东西没什么用, 到现在我也没找到一个应用场景

该方法可以向左(传入负数)或向右(传入正数)旋转, 而跳过一些元素.

ps: 双端队列是线程安全的, 可以在不同的线程中同时从两端利用队列的内容.
